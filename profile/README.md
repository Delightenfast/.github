## 💡 Delight!
개인 맞춤형 가려움증 관리 서비스를 제공하여 삶의 질을 향상시키는 것을 목표로 합니다.

환경요인과 개인의 데이터를 분석하여 최적의 관리 방법을 제공하며, 건강한 피부를 위해 개인 맞춤형 솔루션을 지속적으로 제공합니다.

### 내 손 안의 피부 주치의 "가려움 추적기"
언제 어디서나 24시간 데이터 기반 케어 서비스로 가려움의 원인을 분석해 이를 회피할 수 있도록 돕고자 합니다.
